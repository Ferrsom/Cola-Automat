Small exercise to get familiar with C language based on this exercise:

Programmieren Sie einen Kola-Automaten.
- Zwei parallele const Arrays für Name und Preis der Produkte
- Zwei parallele const Arrays für Taste und Wert der Münzen

Version 1: die Produkte werden angezeigt, dann gibt der Nutzer den Index des gewünschten Produkts ein; die eingegebene Summe wird mit 0 initialisiert; dann werden wiederholterweise die Möglichen Münzen angezeigt und eine Taste erwartet, bei x wird abgebrochen und die eingegebene Summe zurückgegeben (nur die Zahl anzeigen, nicht die Münzen), sonst wird der Wert der Münze zur eingegebenen Summe addiert; wenn diese den Preis des Produkts erreicht, werden das Produkt und das Restgeld ausgegeben.
